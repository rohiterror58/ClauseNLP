INSTALL ALL LIBRARIES IN PROJECT PATH (WINDOWS CMD GUIDE)

Follow the steps below carefully:

--------------------------------------------------
1. Open Command Prompt
--------------------------------------------------
Press:
Windows + R → type cmd → press Enter

--------------------------------------------------
2. Navigate to your project folder
--------------------------------------------------
Use cd command to go to your project directory.

Example:
cd Desktop\your_project_folder

(Replace "your_project_folder" with your actual folder name)

--------------------------------------------------
3. (Recommended) Create a virtual environment
--------------------------------------------------
This keeps dependencies isolated.

Run:
python -m venv venv

--------------------------------------------------
4. Activate the virtual environment
--------------------------------------------------
Run:
venv\Scripts\activate

After activation, you will see (venv) in your terminal.

--------------------------------------------------
5. Install all required libraries
--------------------------------------------------
Make sure requirements.txt is in your project folder.

Run:
pip install -r requirements.txt

--------------------------------------------------
6. If pip is outdated (optional but recommended)
--------------------------------------------------
Run:
python -m pip install --upgrade pip

--------------------------------------------------
7. Special Notes
--------------------------------------------------

• If torch (PyTorch) fails to install:
  Go to: https://pytorch.org
  Select your system and install manually.

• Selenium setup:
  - Make sure Google Chrome is installed.
  - ChromeDriver is handled automatically by webdriver-manager.

• If any module is missing:
  Install manually using:
  pip install module_name

Example:
pip install flask

--------------------------------------------------
8. Run your project
--------------------------------------------------
After installation, run your app:

python app2.py

--------------------------------------------------
DONE!
